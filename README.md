# Desafio 100 Dias de C#

Este repositório faz parte de um desafio pessoal no qual me comprometo a programar em C# durante 100 dias consecutivos.

A proposta é praticar diariamente por meio de exercícios, desafios de lógica e pequenos projetos, com o objetivo de consolidar fundamentos, aprofundar conhecimentos e desenvolver consistência na prática da programação.

---

## Objetivos

- Praticar C# diariamente
- Reforçar lógica de programação
- Aplicar conceitos de Programação Orientada a Objetos
- Explorar recursos do .NET
- Desenvolver pequenos projetos práticos
- Construir um portfólio consistente no GitHub

---

## Como o desafio funciona

- Todos os dias é desenvolvido um exercício ou projeto.
- Cada dia possui sua própria pasta no repositório.
- Os temas evoluem gradualmente em complexidade.
- Sempre que possível, os projetos incluem boas práticas e organização de código.

Os exercícios e projetos podem envolver:

- Aplicações Console
- Estruturas de dados
- Orientação a Objetos
- Manipulação de arquivos
- Consumo de APIs
- Integração com banco de dados
- Conceitos de arquitetura e organização de código

---

## Estrutura do repositório


Cada pasta contém o código-fonte e, quando necessário, explicações sobre o que foi desenvolvido naquele dia.
